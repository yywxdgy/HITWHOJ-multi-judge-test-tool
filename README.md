# HITWHOJ多线程测试工具

- 安装依赖

~~~
npm install
~~~

- 修改目标代码：将 `main.cpp` 替换为要提交的代码。

- 开始测试（参数均为数字）

~~~
python exeute.py [题号] [提交次数]
~~~

---

- 示例：

~~~
python exeute.py 24 5
~~~
